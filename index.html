UV Radiation Detection System







This device is designed to take sun radiation as input through the light sensor.
It is programmed to convert it into a UV index value (from 0-12) which will be displayed and updated on a LCD monitor, along informing the user how much skin protection is needed outside.
The STM32 Microcontroller was programmed in C, using the STM32Cube IDE. The outer structure was designed in AutoCAD and later 3D printed.

Line-Following Automated Rover







This rover is designed and programmed to travel along a path from one location to another.
If the rover drives out of the pathway, it is able to relocate the pathway automatically.
A majority of the structure is 3D printed, along with Analog Line Sensors, DC Motors, and an Arduino.
The line sensors and motors on the wheels are programmed for automation through the Arduino, using C in the Arduino IDE.

Autonomous Truck
Circuit Simulation






This truck simulation was designed and programmed on Autodesk TinkerCAD.

Includes a built-in backing-up alarm system, which has flashing lights and sounds.

The circuit consists of multiple Arduinos (programmed in C), DC Motors, an LED, and a Piezo.
